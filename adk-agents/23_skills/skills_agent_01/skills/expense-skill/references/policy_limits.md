# Expense Policy Limits

| Category | Limit per Claim | Notes |
|-----------------------|-----------------|------------------------------------|
| Meals (individual) | $50 | Receipt required |
| Meals (team/client) | $75 per person | List of attendees required |
| Ground transport | $100 | Uber/Lyft/taxi receipts required |
| Air travel | Economy class | Business class needs VP approval |
| Hotel | $250/night | Higher rates need pre-approval |
| Home office supplies | $200/quarter | Itemised list required |
| Conference fees | $1,000 | Manager pre-approval required |
| Gifts to clients | $50 | Not reimbursable above this limit |

## Special Rules

- **International travel**: Requires 2-week advance approval from Director level.
- **Spouse/partner travel**: Never reimbursable.
- **Parking**: Reimbursable at the actual cost with receipt; valet requires justification.
- **Subscriptions**: Only pre-approved software subscriptions are reimbursable.
